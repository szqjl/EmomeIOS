# EmoFirst iOS 快速开始指南

## 🚀 5分钟快速启动

### 1. 环境检查
```bash
# 检查Flutter版本（需要 >= 3.16.0）
flutter --version

# 检查iOS开发环境
xcode-select --version
```

### 2. 项目初始化
```bash
# 进入项目目录
cd 开发工作流

# 运行自动配置脚本
./scripts/setup_ios.sh

# 或手动执行
flutter pub get
cd ios && pod install && cd ..
```

### 3. 运行项目
```bash
# 在iOS模拟器上运行
flutter run -d ios

# 或指定设备
flutter devices  # 查看可用设备
flutter run -d <device-id>
```

## 🔧 常见问题

### Q: 运行时报错"找不到资源文件"
A: 这是正常的，资源文件需要单独创建。应用仍然可以运行，只是部分功能（动画、音效）可能不完整。

### Q: 如何添加资源文件？
A: 参考 `资源文件创建指南.md` 文档。

### Q: 如何配置iOS签名？
A: 在Xcode中打开 `ios/Runner.xcworkspace`，配置Bundle ID和开发团队。
