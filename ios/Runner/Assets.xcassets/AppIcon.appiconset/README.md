# App图标说明

## 图标设计规范

### 设计概念
- **主元素**: 心电图 + 盾牌组合
- **风格**: 简洁、现代、温暖
- **颜色**: 情感红色 (#FF6B6B) + 白色/浅色背景

### 尺寸要求

需要准备以下尺寸的图标：

| 尺寸 | 用途 | 必需 |
|------|------|------|
| 1024x1024 | App Store | ✅ |
| 180x180 | iPhone主屏幕（3x） | ✅ |
| 120x120 | iPhone主屏幕（2x） | ✅ |
| 87x87 | iPhone设置（3x） | ✅ |
| 58x58 | iPhone设置（2x） | ✅ |
| 80x80 | iPhone Spotlight（2x） | ✅ |
| 120x120 | iPhone Spotlight（3x） | ✅ |
| 76x76 | iPad主屏幕（2x） | ✅ |
| 152x152 | iPad主屏幕（2x） | ✅ |
| 167x167 | iPad Pro主屏幕（2x） | ✅ |

### 设计工具
- **Figma**: 推荐使用Figma设计
- **Sketch**: 也可以使用Sketch
- **在线工具**: https://www.appicon.co/

### 导出步骤
1. 设计1024x1024px的主图标
2. 使用工具自动生成所有尺寸
3. 将图标文件放入此目录
4. 更新 `Contents.json` 中的路径

### 临时方案
在图标未准备好之前，可以使用：
- Flutter默认图标
- 纯色背景 + 文字"EF"
- 简单的红色圆形
