## 需要在Xcode中完成的配置

### 1. Bundle Identifier
```
1. 打开 Xcode
2. 选择项目 -> Runner -> General
3. 设置 Bundle Identifier: com.emofirst
```

### 2. 部署目标
```
1. 在 General 标签页
2. 设置 iOS Deployment Target: 14.0
```

### 3. 签名和功能
```
1. 在 Signing & Capabilities 标签页
2. 选择开发团队
3. 启用自动签名
```

### 4. 添加App图标
```
1. 打开 Assets.xcassets
2. 选择 AppIcon
3. 拖拽图标文件到对应尺寸位置
```

## 常见问题

### Q: 为什么不需要网络权限？
A: EmoFirst采用本地处理，所有数据仅在内存中，不涉及网络传输。

### Q: 触觉反馈需要权限吗？
A: 不需要，iOS系统内置的Haptic Feedback无需权限。

### Q: 音频播放需要权限吗？
A: 不需要，播放本地音频文件无需权限。
