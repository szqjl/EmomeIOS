# 动画资源说明

## 需要的动画文件

### heart_break.json
**Lottie动画文件 - 心形碎裂动画**

#### 动画规格
- **时长**: 2.3秒
- **帧率**: 60fps
- **尺寸**: 150x150px
- **格式**: Lottie JSON
- **内存占用**: < 200KB

#### 动画时间线
```
0.0-0.5秒：屏幕中央出现完整的心形图标
0.5-1.0秒：心形开始出现裂痕
1.0-1.8秒：裂痕扩散，颜色从#FF6B6B渐变为#FF3838
1.8-2.3秒：心形碎成12个主要碎片，向四周轻微扩散
```

#### 创建方法
1. 使用 After Effects 创建动画
2. 使用 Bodymovin 插件导出为 Lottie JSON
3. 或使用在线工具：https://lottiefiles.com/

#### 颜色规范
- 初始颜色: #FF6B6B (情感红色)
- 碎裂颜色: #FF3838 (深红色)
- 背景: 透明

#### 同步效果
- 音效：低沉的"压力积聚"声（1.5秒）
- 震动：iOS中等强度（1.8秒触发）
