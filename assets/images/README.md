# 图片资源说明

## 当前设计

EmoFirst 采用**纯色背景 + 动画**的设计方案，**不需要图片资源**。

### 颜色方案
- 背景色: #F5F7FA (柔和的灰白色)
- 主按钮: #FF6B6B (情感红色)
- 呼吸引导: 从红色渐变到蓝色

### 图标
- 使用Emoji表情符号（😠😢💔😶🌀😓）
- 使用系统字体图标
