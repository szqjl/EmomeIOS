# 音频资源说明

## 需要的音频文件

### 1. pressure_build.mp3
**压力积聚音效**
- **时长**: 1.5秒
- **音量**: 60%
- **用途**: 情绪启动动画同步音效
- **特点**: 低沉的、逐渐增强的压力感

### 2. bubble_pop_1.mp3 / bubble_pop_2.mp3 / bubble_pop_3.mp3
**泡泡破碎音效（3个变体）**
- **时长**: 0.3-0.5秒
- **音量**: 60%
- **用途**: 释放阶段音效选项A
- **特点**: 清脆、短促的泡泡破裂声
- **播放频率**: 每次点击播放一个随机泡泡音

### 3. wind_chime_1.mp3 / wind_chime_2.mp3 / wind_chime_3.mp3
**风铃音效（3个音符）**
- **时长**: 1-2秒
- **音量**: 50%
- **用途**: 释放阶段音效选项B
- **特点**: 空灵、舒缓的风铃声
- **播放频率**: 每3次点击播放一个音符（C/E/G序列）

### 4. fire_crackle.mp3
**篝火星音效**
- **时长**: 0.5-1秒
- **音量**: 40%
- **用途**: 释放阶段音效选项C
- **特点**: 温暖、轻微的"噼啪"声
- **播放频率**: 每次点击播放

## 音频获取方式

### 方案1：免费音频库
- **Freesound.org**: https://freesound.org/
- **Zapsplat**: https://www.zapsplat.com/
- **Adobe Stock**: 免费试用

### 方案2：AI生成
- **Mubert**: AI生成音效
- **AIVA**: AI音乐生成

### 方案3：录制
- 使用手机录音
- 使用Audacity编辑
- 导出为MP3格式

---

## 音频格式要求

- **格式**: MP3
- **采样率**: 44.1kHz
- **比特率**: 128kbps
- **声道**: 单声道或立体声
- **文件大小**: 每个文件 < 500KB
